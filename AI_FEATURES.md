# 🤖 换换校园 - AI 功能说明

## AI 能力架构

本平台深度集成了阿里百炼 AI 服务（kimi-k2.5 模型），提供以下智能功能：

---

## 1. AI 智能估价 💰

**功能描述：**
- 用户输入物品名称、描述、分类、成色
- AI 分析市场行情，给出专业估价
- 提供估价区间、市场分析、影响因素、出售建议

**使用场景：**
- 发布物品时不知道定价
- 想了解物品的市场价值
- 获取专业的出售建议

**API 端点：**
```
POST /api/v1/ai/evaluate
```

---

## 2. AI 对话式发布 📝

**功能描述：**
- 用户用自然语言描述物品
- AI 自动提取关键信息
- 生成完整的发布内容（标题、描述、分类、标签等）

**示例：**
```
用户输入："我有个用了半年的iPad Pro 11寸，128G，银色，保护得很好，想换个相机"

AI 输出：
- 标题：iPad Pro 11寸 128G 银色
- 描述：使用半年，保护良好，功能正常...
- 分类：电子产品
- 成色：95新
- 期望换到：相机
- 智能标签：平板、苹果、数码
```

**API 端点：**
```
POST /api/v1/ai/parse-item
```

---

## 3. AI 智能匹配 🔍

**功能描述：**
- 分析用户的物品和交换需求
- 从平台物品中智能匹配最佳交换对象
- 给出匹配分数和推荐理由

**使用场景：**
- 快速找到合适的交换对象
- 提高交换成功率
- 发现潜在的交换机会

**API 端点：**
```
POST /api/v1/ai/find-matches
```

---

## 4. AI 智能客服 💬

**功能描述：**
- 7×24 小时在线自动回答
- 解答平台使用问题
- 提供交换建议和交易指导

**支持的问题类型：**
- 如何发布物品？
- 怎么进行交换？
- 交换安全吗？
- 如何认证学生身份？
- 什么物品不能交换？

**API 端点：**
```
POST /api/v1/ai/customer-service
```

---

## 5. AI 内容审核 🛡️

**功能描述：**
- 自动审核物品描述
- 识别违禁品和违规内容
- 检测欺诈行为

**审核维度：**
- 违禁品（毒品、武器等）
- 违规服务（代考、代写等）
- 敏感政治内容
- 色情低俗内容
- 诈骗、虚假广告
- 食品、药品（校园内禁止）

**API 端点：**
```
POST /api/v1/ai/content-review
```

---

## 6. AI 交易助手 🤝

**功能描述：**
- 生成交易话术
- 价格谈判建议
- 交换方案推荐

**API 端点：**
```
POST /api/v1/ai/generate-message
POST /api/v1/ai/negotiate-advice
```

## 使用流程

### 发布物品（AI 辅助）
1. 点击首页 "AI智能发布" 或发布页的 AI 入口
2. 用自然语言描述物品
3. AI 自动生成完整发布信息
4. 确认或修改后发布

### 物品估价
1. 在发布页点击 "AI估价" 入口
2. 填写物品信息
3. 获取 AI 估价报告
4. 使用建议价格

### 智能客服
1. 点击首页 "AI助手"
2. 输入问题或选择快捷问题
3. 获取 AI 回答
